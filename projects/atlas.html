Challenge

1

Create a simple user interface for searching for files in the existing machine that returns data as quickly as possible without hardware limitations.

2

Secure application from XSS (Cross-site scripting) and external access.

Solution

1

Optimizing File Search by implementing efficient searching algorithms, combining Binary Search and Quick sort to establish an efficient return.

2

Use Tauri's Permissions to prevents access from outside the application and sanitize data on crucial elements.

Outcome

The application successfully allows the user to browse the existing state of the file system with permissions with simple functions of a file manager being implemented using CRUD, and applied optimizations for quick data retrievals providing the availability for use on limiting hardware.
